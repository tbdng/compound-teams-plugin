# Compound Teams Marketplace

A Claude Code plugin marketplace distributing the **compound-teams** plugin.

## Install

```
/plugin marketplace add https://github.com/tbdng/compound-teams-plugin
/plugin install compound-teams
```

Then in any project:
```
/ct:setup
```
